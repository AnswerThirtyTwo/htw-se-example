.Formatierung von Text
[cols="1,1,2"] 
|===
|Formatierung |Syntax |Beschreibung

|*bold*
|\*bold*
|fetter Text

|_italic_
|\__italic__
|kursiver Text

|monospace
|`monospace`
|Text mit fester Laufweite


|*_bolditalic_*
|\\*_bolditalic_*
|Text mit kombinierter Formatierung

|\\**bold**__italic__
|\\**bold**__italic__
|Text mit Formatierung innerhalb eines Wortes

|===
